kSqlExpression

An SQL WHERE clause expression builder.

This works by building up the expression as a hierarchical tree of nodes. The toString() method then traverses this tree in order to build the final expression string.

All the build methods in this object return the object instance for chained method calling purposes.

Instance Methods

constructor()

Initialise the expression.

begin()

Begin a compound expression.

A compound expression is an expression which contains an intersection (AND) or union (OR) of one or more sub expressions. When rendered it will be fully contained with brackets.

end()

End the current compound expression.

and(expr)

Combine the current expression with the given expression using the intersection operator (AND).

or(expr)

Combine the current expression with the given expression using the union operator (OR).

toString()

Get the final fully constructed expression string.
